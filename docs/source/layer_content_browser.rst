Layer Content Browser
---------------------

Displays current content from USD layers (regardless of format or if they've saved on disk).
Array attributes and time samples have their contents reduced to a maximum of 6 entries.

The following example browses the contents of the layers that modify the ``displayColor`` property of the ``ChairB_2`` (as modified above).
